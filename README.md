# reactive-psc

React.js View >>= RX Binding >>= Purescript Controller.

## Required Tooling:
 - [psc/psc-make](http://www.purescript.org/)
 - [npm](http://nodejs.org/)
 - [bower](http://bower.io/)
 - [grunt](http://gruntjs.com/)

## Building

```
npm install
bower update
grunt
grunt browser
```

Point your browser at ** localhost:9001/static **
